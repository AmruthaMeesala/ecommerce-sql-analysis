# E-Commerce Sales SQL Analysis Project
## Project Overview
This project analyzes e-commerce sales data using PostgreSQL.
The project includes:
* Joins
* Aggregate functions
* Subqueries
* CTEs
* Window functions
* Ranking functions
* Customer segmentation
* Revenue analysis

## Database Tables
* customers
* products
* orders
* payments
* reviews

## Key Analysis Performed
* Total revenue analysis
* Top customers
* Product ranking
* Customer segmentation
* Running revenue totals
* Payment analysis
* Window function analysis using:
  * RANK()
  * ROW_NUMBER()
  * LAG()
  * LEAD()

## SQL Concepts Used
* INNER JOIN
* LEFT JOIN
* GROUP BY
* HAVING
* CASE WHEN
* CTE
* Window Functions
* Aggregate Functions
* Subqueries

* PostgreSQL
* DB Fiddle
* GitHub
